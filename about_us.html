About Us

GTU Vidhyarthi is a software application wherein the GTU enrolled (affiliated) students have to provide login credentials to use the features of the app. A student has to surf through different websites to check result, GTU circulars, student portal, design engineering, 100 activity points, reference materials, previous years’ GTU papers and for what not. ‘GTU Vidhyarthi’ is a user-friendly app with a classic graphical user interface provides easy navigation to all the above mentioned websites, face recognition encryption, latest circulars & result updates with notifications right away, blogging facilities for the alumni to provide guidance to the junior students and many more features with a very simple login system consisting of enrollment number and password.
